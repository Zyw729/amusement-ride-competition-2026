# 光轴布局尺寸计算

## 核心约束

光轴是**竖向贯穿**的固定杆（从顶部大圆盘→地面），不能与任何旋转部件（桁架、胶囊）碰撞。

## 碰撞分析

### 旋转体扫略范围

| 部件 | 径向范围 | 说明 |
|------|---------|------|
| 塔柱 | r = 0 ~ 2.12m | □3m×3m对角线半长 |
| 内环胶囊(面朝外) | r = 5.25 ~ 6.75m | 中心r=6m, 舱深1.5m |
| 外环胶囊(面朝内) | r = 7.25 ~ 8.75m | 中心r=8m, 舱深1.5m |
| 两环交战间隙 | r = 6.75 ~ 7.25m | **仅0.5m** |

### 内环光轴位置

必须位于内环胶囊扫略范围以内：r < 5.25m
必须不碰塔柱：r > 2.12m

**选定：r_inner = 3.5m** (距塔柱角点1.38m, 距胶囊内缘1.75m) ✅

### 外环光轴位置

是否可放在 r=6.75~7.25m 间隙内？光轴直径~0.15m → 需0.5m净宽 → 间隙仅0.5m → **太紧** ❌

**选择：放在外环外面（r > 8.75m）**

结合回转支承安装需求：**选定 r_outer = 9.5m** (胶囊外缘外0.75m) ✅

### 内/外光轴间距

9.5 - 3.5 = **6.0m** → 充足，不会相聚过近 ✅

## 最终尺寸表

```
俯视径向截面 (从外到内):

  r(m)   部件
  ────   ──────────────────────────────
  11.0   安全围栏
  10.0   顶部大圆盘边缘 (~φ20m)
   9.5   █ 外环光轴 ×4 (固定) + 外环回转支承
   9.0   外环桁架外缘
   8.5   外环桁架内缘
   8.0   ◉ 外环胶囊中心, 面朝内 →
   7.25  外环胶囊内缘
  ────   ═══ 交战间隙 (0.5m) ═══
   6.75  内环胶囊外缘
   6.0   ◉ 内环胶囊中心, ← 面朝外
   5.5   内环桁架外缘
   4.0   内环桁架内缘
   3.5   █ 内环光轴 ×4 (固定) + 内环回转支承
   2.12  塔柱□3×3对角线半长
     0   ⊙ 塔柱中心
```

## 均布角度

| 光轴组 | 数量 | 均布半径 | 角度间隔 | 相邻弦距 |
|--------|------|---------|---------|---------|
| 内环 | 4根 | 3.5m | 90° | 4.95m |
| 外环 | 4根 | 9.5m | 90° | 13.44m |

建议内/外光轴错开45°，使最近的内外光轴距离最大化：
```
最近距离 = √(9.5² + 3.5² - 2×9.5×3.5×cos(45°))
        = √(90.25 + 12.25 - 47.06)
        = √55.44
        = 7.45m  ✅ 非常安全
```

## 干涉校验

| 检查项 | 间隙 | 判定 |
|--------|------|------|
| 内光轴 ↔ 塔柱角点 | 3.5-2.12=1.38m | ✅ |
| 内光轴 ↔ 内环桁架内缘 | 4.0-3.5=0.5m | ✅ |
| 内光轴 ↔ 内环胶囊内缘 | 5.25-3.5=1.75m | ✅ |
| 外光轴 ↔ 外环桁架外缘 | 9.5-9.0=0.5m | ✅ (轴承界面) |
| 外光轴 ↔ 外环胶囊外缘 | 9.5-8.75=0.75m | ✅ |
| 内外光轴最近距离 | 7.45m | ✅ |
| 顶部大圆盘直径 | φ20m | 可接受 |

---

## 结论

内环光轴均布半径 **r=3.5m**，外环光轴均布半径 **r=9.5m**，错开45°布置。

内外光轴相距 **6m（径向）~7.45m（空间最近）**，不会相聚过近。所有旋转体与固定光轴均有 ≥0.5m 安全间隙。
