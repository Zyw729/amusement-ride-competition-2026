## 2. 运动学 (3rpm)

| 参数 | 内环 | 外环 |

### 2.2 向心加速度

```
内环: a_n = ω² × R = 0.314² × 6.0 = 0.59 m/s² = 0.06g
```

## 3. 动力学与功率

### 3.1 升降驱动 (顶部电机+滑轮)

**内环 (4台电机, 每台承载1/4载荷):**

配重65%后单缆张力：
```
F_cable = (15,700 × 9.81 × 0.35) / (4 × 0.9)
        = 53,905 × 0.35 / 3.6
        = 5,245 N
```
实际上4台电机各负责1根缆:
```
单缆载荷 = 15,700 × 9.81 × (1 - 0.65) / 4
         = 154,017 × 0.35 / 4
         = 13,476 N

正常 2m/s: P = 13,476 × 2 / 0.88 = 30.6 kW → 40kW电机 ✅
终局 4m/s: P = 13,476 × 4 / 0.88 = 61.3 kW → 40kW过载153% 短时可行
```

**外环 (4台电机):**
```
单缆载荷 = 24,800 × 9.81 × 0.35 / 4 = 21,297 N
正常 2m/s: P = 21,297 × 2 / 0.88 = 48.4 kW → 50kW电机 ✅
终局 4m/s: P = 21,297 × 4 / 0.88 = 96.8 kW → 50kW过载194% 短时可行
```

**结论: 内环4×40kW + 外环4×50kW = 360kW (升降总装机)**

### 3.2 旋转驱动 (导套上电机→小齿轮→齿圈)

**转动惯量:**

内环 (近似圆环 r=5m):
```
I_inner = 15,700 × 5² = 392,500 kg·m²
```

外环 (近似圆环 r=9m):
```
I_outer = 24,800 × 9² = 2,008,800 kg·m²
```

**加速 (0→3rpm 用 8s):**

```
α = 0.314/8 = 0.0393 rad/s²

内环: τ_acc = 392,500 × 0.0393 = 15,400 N·m
      P_peak = 15,400 × 0.314 = 4,840 W ≈ 4.8 kW (总)
      每台 = 1.2 kW

外环: τ_acc = 2,008,800 × 0.0393 = 78,900 N·m
      P_peak = 78,900 × 0.314 = 24,800 W ≈ 24.8 kW (总)
      每台 = 6.2 kW
```

加上齿轮啮合效率(0.92)和轴承摩擦:
```
内环每台: 1.2/0.92 + 摩擦 ≈ 2 kW
外环每台: 6.2/0.92 + 摩擦 ≈ 8 kW
```

**15kW和30kW选型非常充裕**，大部分时间在低负载运行。

## 4. 结构校核

### 4.1 光轴校核

**外光轴(最不利, φ180mm, r=9.5m):**

光轴承受导套传递的径向力（环体偏载/风载）和轴向力（不承受, 仅导向）。

径向力估算(满载偏心+风载): F_radial ≈ 50 kN/根

简化为两端铰支杆(L=50m):
```
I = π×0.18⁴/64 = 5.15×10⁻⁵ m⁴
P_cr = π²×206×10⁹×5.15×10⁻⁵ / 50² = 41,700 N = 41.7 kN
```

> 41.7 kN < 50 kN — 单根光轴屈曲载荷不足！需要加大直径或增加中间支撑。

**修正：外光轴改为φ250mm**
```
I = π×0.25⁴/64 = 1.92×10⁻⁴ m⁴
P_cr = π²×206×10⁹×1.92×10⁻⁴ / 50² = 155,600 N = 155.6 kN
安全系数 = 155.6/50 = 3.1 ✅
```

**内光轴 φ150mm:**
```
I = π×0.15⁴/64 = 2.49×10⁻⁵ m⁴
P_cr = π²×206×10⁹×2.49×10⁻⁵ / 50² = 20,200 N = 20.2 kN
```
径向力较小(~25kN), 20.2 < 25, 需加大。

**内光轴改为φ180mm**: I = 5.15×10⁻⁵, P_cr = 41.7 kN, 安全系数 41.7/25 = 1.67 (偏低)。

**内光轴改为φ200mm**: I = 7.85×10⁻⁵, P_cr = 63.6 kN, 安全系数 63.6/25 = 2.5 ✅
