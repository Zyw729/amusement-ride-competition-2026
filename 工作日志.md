# 工作日志 — 游乐设施创意赛2026

> 赛题：「青年刺激」大型游乐设施创意设计
> 团队：中南大学，机械设计制造及其自动化

---

## 2026-05-25 第一次讨论

### 确定方向

- 用户背景：大三，机械设计制造及其自动化。SolidWorks建模 + ANSYS静力学 + ADAMS动力学，队友KeyShot渲染。编程仅基础。
- 赛题选定：**赛题一「青年刺激」**。
- 用户初始想法：过山车/海盗船/大摆锤已过时，想要多感官、有速度、类似"飞行员驾驶战斗机"的创新设施。

### 方案筛选

Claude 提出 5 个概念方案：

| 方案 | 名称 | 核心 |
|------|------|------|
| A | 天袭者 | 可变姿态过山车 |
| B | 双子星 | 双轨交互竞技过山车 |
| C | 深渊之门 | 三段介质穿梭（空→水→空） |
| D | 暴风眼 | 多感官风暴模拟塔 |
| E | 幻影骑士 | 俯卧式空战模拟器 |

用户选择 **B + E（多人竞技 + 战斗沉浸）**。

### 核心约束

⚠️ **经济性约束**：用户明确指出单座竞技（如1v1战斗机模拟器）载客量低、成本回收周期长，必须做到多人同乘才能商业化。过山车类一次20-32人的载客效率是参照标杆。

### 融合方案输出

Claude 提出两个多人竞技融合方案：

**方案一：「天袭中队」双环竞技跳楼机** ⭐ 推荐
- 中央塔 + 双独立环（红蓝两队），每环8-10个双人胶囊舱
- 总载客32-40人/次
- 双环反向旋转+独立升降，胶囊俯仰联动
- 激光竞技 + 胜败差异化体验

**方案二：「宿敌」双轨交互竞技过山车**
- 双轨并行，多段物理交互
- 总载客40-48人/次
- LSM电磁弹射，团队协作蓄能

### 选择结论

**选定方案一「天袭中队」**。理由：
1. 机构类型在赛事中少见（全新，非过山车改款）
2. 三层递进结构（塔→环→舱），每层可做静力学+动力学分析
3. 天然适合ADAMS多体动力学仿真
4. KeyShot渲染效果震撼（夜景科幻风格）
5. 保留了"编队空战"的幻想基因

## 2026-05-25 第二次讨论（参数确定 + 机构设计）

### 参数确认

| 参数 | 确定值 |
|------|--------|
| 塔高 | **50m** |
| 载客 | **双环 × 10舱 × 2人 = 40人/次** |
| 竞技 | **基础版**（激光对射积分 + 胜方冲顶/败方急降） |
| 姿态 | **俯卧式**（脸朝下趴着，类似战斗机飞行员） |

### 俯卧式说明

- 两人并排趴卧在胶囊舱中
- 透明钢化玻璃底板 → 直面俯瞰地面
- 五点式背部安全带 + 腿部液压压杆
- 双手握把集成激光扳机
- 参考：伊卡洛斯跳楼机（95m俯卧坠落）、创极速光轮（前倾骑坐）

### 完整机构原理设计

输出文件：**`设计方案_天袭中队.md`**（约600行）

涵盖内容：
1. 总体概述 + ASCII结构图（正视图/俯视图）
2. 技术参数汇总（总体/运动/加速度/电力）
3. 结构系统设计（塔体/环体/胶囊舱/安全约束）
4. 驱动与传动系统（升降卷扬机/回转支承旋转/伺服俯仰）
5. 制动与安全系统（四级制动+监测传感器+安全联锁）
6. 竞技系统（激光对射+积分反馈+运行流程时序）
7. 运动学与动力学分析框架（坐标系+运动方程+ADAMS建模+ANSYS工况）
8. 创新点总结（结构/功能/体验三维度）
9. 与现有设施对比表
10. 附录：SolidWorks建模建议（装配体层级结构）
