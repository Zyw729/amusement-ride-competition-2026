## 2026-05-25 第一次讨论

### 确定方向

- 用户背景：大三，机械设计制造及其自动化。SolidWorks建模 + ANSYS静力学 + ADAMS动力学，队友KeyShot渲染。编程仅基础。
- 赛题选定：**赛题一「青年刺激」**。
- 用户初始想法：过山车/海盗船/大摆锤已过时，想要多感官、有速度、类似"飞行员驾驶战斗机"的创新设施。

### 方案筛选

Claude 提出 5 个概念方案：

| 方案 | 名称 | 核心 |
|------|------|------|
| A | 天袭者 | 可变姿态过山车 |
| B | 双子星 | 双轨交互竞技过山车 |
| C | 深渊之门 | 三段介质穿梭（空→水→空） |
| D | 暴风眼 | 多感官风暴模拟塔 |
| E | 幻影骑士 | 俯卧式空战模拟器 |

用户选择 **B + E（多人竞技 + 战斗沉浸）**。

### 核心约束

⚠️ **经济性约束**：用户明确指出单座竞技（如1v1战斗机模拟器）载客量低、成本回收周期长，必须做到多人同乘才能商业化。过山车类一次20-32人的载客效率是参照标杆。

### 融合方案输出

Claude 提出两个多人竞技融合方案：

**方案一：「天袭中队」双环竞技跳楼机** ⭐ 推荐
- 中央塔 + 双独立环（红蓝两队），每环8-10个双人胶囊舱
- 总载客32-40人/次
- 双环反向旋转+独立升降，胶囊俯仰联动
- 激光竞技 + 胜败差异化体验

**方案二：「宿敌」双轨交互竞技过山车**
- 双轨并行，多段物理交互
- 总载客40-48人/次
- LSM电磁弹射，团队协作蓄能

### 选择结论

**选定方案一「天袭中队」**。理由：
1. 机构类型在赛事中少见（全新，非过山车改款）
2. 三层递进结构（塔→环→舱），每层可做静力学+动力学分析
3. 天然适合ADAMS多体动力学仿真
4. KeyShot渲染效果震撼（夜景科幻风格）
5. 保留了"编队空战"的幻想基因

## 2026-05-25 第四次讨论（结构深化 + v2升级）

### 关键问题发现与解决

**问题1：上下环会互相挡住**
- 原方案：两环上下排列, 下环无法升到上环以上 → 运动范围受限
- 解决：改为**同心内外环** (内环φ12m红队 + 外环φ16m蓝队)
- 两环在同一高度对峙, 内环可自由穿过外环中心运动

**问题2：不能看屁股**
- 解决：内环胶囊径向朝外 + 外环胶囊径向朝内 → 天然面对面
- 偏航追踪±30° → 持续正对
- 相对角速度96°/s → 每0.375秒一对胶囊正对

**问题3：交战太短**
- 解决：**弱者补偿机制**——分差自动调节环体高度
- 高空有攻击加成但被弹面积大, 低空反之
- ≤20分不干预, 20-50分±5m补偿, >50分强制平衡

**问题4：角色分工**
- 每舱2人：**射手**（双扳机激光进攻）+ **驾驶员**（操纵杆+节流阀防守）
- 40人全员参与, 无人被动乘坐

### 胶囊-环体连接机构

设计两级铰接：偏航(Yaw)在上 → 俯仰(Pitch)在下
- 偏航: 750W伺服 + 行星减速i=50 + 失电制动器
- 俯仰: 1.5kW伺服 + 蜗轮蜗杆i=120 (自锁)
- 失效保护: 断电→偏航弹簧锁死+俯仰蜗轮自锁
