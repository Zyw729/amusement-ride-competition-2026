> 赛题一：「青年刺激」大型游乐设施创意设计
> 目标群体：15-25岁青年

## 1. 总体概述

### 1.1 设施简介

"谁与争锋"是一款融合**编队竞技**与**自由落体**的大型游乐设施。中央塔柱 □3m×3m ×50m 支撑顶部大圆盘(φ~20m)，圆盘搭载全部电机与滑轮。8根独立光轴（内4+外4）从上到下贯穿，分别导向内外环。内环(红队 φ12m)与外环(蓝队 φ16m)在同一高度对峙，各载10个双人胶囊舱（共40人/次）。旋转采用齿轮-齿圈啮合驱动（类似挖掘机回转），转速3rpm。竞技积分自动触发弱者补偿，终局胜方冲顶、败方急降。

### 1.2 核心创新

- **顶部大圆盘集中动力**：8组电机+滑轮均置于顶部平台，缆绳垂直下放，结构简洁
- **独立光轴导向**：8根光轴均布，取代传统塔身导轨，两环互不干涉
- **齿轮旋转驱动**：电机→小齿轮→外齿圈（类似挖掘机回转），与升降完全解耦
- **角色分工**：射手(进攻)+驾驶员(防守)，40人全员参与
- **弱者补偿**：分差自动调节高度，比赛始终胶着

### 1.4 情境主题：「苍穹对决」

"谁与争锋"设定未来竞技世界观：两艘母舰各率一支飞行编队，在万米高空展开荣誉对决。乘客化身为红/蓝战队飞行员，以俯卧战斗姿态驾驭胶囊战机，在旋转升降的立体战场中激光对射、协同攻防。

**叙事线：**
- **登舰**（上下客）→ 母舰停泊，飞行员登机就位
- **升空**（爬升预热）→ 编队同步升空至交战高度，旋转加速，武器系统预热
- **鏖战**（战斗阶段）→ 双环独立升降+反向旋转，弱者补偿确保战局始终胶着
- **决胜**（终局）→ 积分锁定，胜方冲顶"凯旋"，败方急降"坠毁"
- **返航**（返航）→ 编队缓降归舰，排名公布

**多感官氛围：**
- 视觉：红蓝激光束穿梭 + 舱内HUD瞄具屏 + 塔体LED光带随积分变色
- 听觉：背景战斗音效 + 激光命中提示音 + 终局高潮配乐
- 触觉：俯卧失重感 + 俯仰G值变化 + 被锁定时舱体震动反馈
- 观景：透明玻璃底板直面50m高空地面，胶囊旋转时环瞰全景

### 3.4 导套 + 回转支承 + 环体

每个环的组成（从上到下）：

```
    光轴 (固定)
        │
    ┌───┴───┐
    │ 导套    │  ← 直线轴承沿光轴滑动, 不旋转
    │ + 电机  │  ← 旋转电机(带小齿轮)安装在导套上
    │ + 小齿轮 │
    └───┬───┘
        │
    ┌───┴───┐
    │ 回转支承│  ← 内圈固定于导套 / 外圈带齿形
    │ (大轴承)│     齿圈与小齿轮啮合
    └───┬───┘
        │
    ┌───┴───┐
    │ 环体桁架│  ← 连接回转支承外圈, 可旋转
    │ + 悬臂  │     悬臂末端连接胶囊舱
    │ + 胶囊  │
    └───────┘
```

**关键点**：导套只沿光轴升降，不旋转。旋转发生在回转支承处——导套上的电机驱动小齿轮，小齿轮啮合回转支承外圈的齿形，带动环体旋转。与挖掘机上车体回转原理相同。

- 旋转时环瞰四周，升降时感受高度剧烈变化，敌方胶囊近距离掠过时视觉冲击极强

### 3.8 胶囊铰接机构（偏航+俯仰）

胶囊通过两级铰接连接悬臂末端：偏航(Yaw)在上 → 俯仰(Pitch)在下。

| 关节 | 驱动 | 减速器 | 自锁 | 安全 |
|------|------|--------|------|------|
| 偏航 ±30° | 750W AC伺服 | 行星 i=50 | ❌ | 失电制动器 |
| 俯仰 ±30° | 1.5kW AC伺服 | 蜗轮蜗杆 i=120 | ✅ | 自锁+失电制动器 |

失效保护：断电 → 偏航制动器锁死 + 俯仰蜗轮自锁 → 舱体固定。

| 钢缆 | 6×36WS-IWRC, 1960MPa | 破断764kN，安全系数>20 |
| 配重 | HT250 铸铁块 | 低成本，高密度(7.2t/m³) |

## 4. 驱动系统

### 4.1 升降驱动

- 每环4台电机+滑轮，置于顶部大圆盘上
- 钢缆 φ28mm 垂直下放（无侧面走缆），连接导套吊点
- 配重：每环4组配重块（沿光轴附件或缆绳并行布置），配重比65%
- 变频调速 0-4m/s

### 4.2 旋转驱动

- 每环4台电机带小齿轮，安装在导套上
- 小齿轮与回转支承外齿圈啮合
- 电机旋转 → 小齿轮 → 齿圈 → 回转支承外圈 → 环体旋转
- 导套不转，只有回转支承以下旋转

### 4.3 升降与旋转完全解耦

升降和旋转是两套独立系统：
- 升降：顶部电机→缆绳→导套沿光轴上下
- 旋转：导套上的另一个电机→小齿轮→齿圈→环体旋转

两者可同时进行，互不影响。

### 4.4 电气控制系统

**控制架构：**

| 层级 | 功能 | 实现 |
|------|------|------|
| 主控 | 全局运行调度、安全联锁 | 双冗余PLC（西门子S7-1500） |
| 驱动控制 | 升降/旋转/俯仰伺服 | 变频器（升降）+ 伺服驱动器（旋转/俯仰） |
| 竞技系统 | 激光计分、弱者补偿、排名 | 工控机 + 激光传感器阵列 |
| 安全回路 | 急停、超速、越位、失电制动 | 硬件安全继电器回路（独立于PLC） |
| 人机界面 | 操作员监控、维护诊断 | 双屏操作台 + 远程监控 |
| 灯光 | 塔体LED矩阵 + 环体光带 | DMX512协议，随竞技状态联动 |

**安全回路设计（独立硬件）：**
- 多个急停按钮（操作台×2 + 站台×4 + 维修平台×2）→ 串联安全继电器
- 超速检测：编码器脉冲频率 → 超阈值 → 安全继电器断开 → 失电制动器动作
- 越位检测：上下极限限位开关（机械式+电子式双冗余）
- 安全PLC与主控PLC交叉心跳检测，任一方异常即全线停机

## 5. 竞技系统

### 5.1 面对面机制

- 内环胶囊径向朝外 + 外环胶囊径向朝内 → 天然面对面
- 两环反向 3rpm → 相对角速度 36°/s → 每0.5秒正对
- 偏航追踪 ±30° 确保持续对视

### 5.2 弱者补偿

| 分差 | 系统干预 |
|------|---------|
| ≤15分 | 不干预 |
| 15-35分 | 落后方 ±5m 高度补偿 |
| >35分 | 落后方升入高空优势区，领先方降至低空 |

终局前30秒锁定高度。

### 5.3 运行流程

| 阶段 | 时长 | 高度 | 旋转 | 竞技 |
|------|------|------|------|------|
| 上下客 | ~90s | 5m | 静止 | 校验 |
| 爬升 | ~30s | 5→25m | 加速至3rpm | 练习(不计分) |
| 战斗 | ~90s | 20-40m 独立升降 | 3rpm | 正式积分 |
| 终局 | ~15s | 胜→48m / 败→8m | 持续 | 锁定 |
| 返航 | ~30s | 缓降至5m | 减速 | 排名 |
