## 2. 运动学计算

### 2.1 胶囊绝对运动方程

胶囊舱 i (在环上固定角 α_i = 36°×(i-1), i=1..10) 的全局坐标：

```
x_i(t) = R·cos(θ(t) + α_i + ψ_i(t))
y_i(t) = R·sin(θ(t) + α_i + ψ_i(t))
z_i(t) = z_ring(t) + L·sin(β_i(t))

其中: R = 环半径 (内6m / 外8m)
      θ = 环旋转角
      ψ_i = 胶囊偏航角 (±30°)
      β_i = 胶囊俯仰角 (±30°)
      z_ring = 环体高度
      L = 悬臂长 + 胶囊半高 (内~3m / 外~5.5m)
```

**速度：**
```
v_xi = -R·(ω+ω_ψ)·sin(θ+α_i+ψ_i)
v_yi =  R·(ω+ω_ψ)·cos(θ+α_i+ψ_i)
v_zi = v_z + L·ω_β·cos(β_i)

其中: ω = dθ/dt (环旋转角速度)
      ω_ψ = dψ/dt (偏航角速度)
      v_z = dz/dt (升降速度)
      ω_β = dβ/dt (俯仰角速度)
```

**加速度：**
```
a_径向 = R·(ω+ω_ψ)²                          ← 向心加速度
a_切向 = R·α_rot                              ← 角加速度贡献
a_垂直 = a_z + L·α_β·cos(β) - L·ω_β²·sin(β)  ← 垂直加速度
```

### 2.2 战斗阶段向心加速度

8 rpm = 0.838 rad/s:

| | 内环 (R=6m) | 外环 (R=8m) |
|---|---|---|
| 向心加速度 a_n | 0.838²×6 = **4.21 m/s²** | 0.838²×8 = **5.62 m/s²** |
| 换算为 g | **0.43g** | **0.57g** |

> 外环乘客离心力略大于内环。但外环悬臂更长 → 更刺激的"被甩出去"感，天然形成两队体验差异。

### 2.3 相对角速度 (面对面频率)

内环↻ 48°/s + 外环↺ 48°/s (均以8rpm计):
```
ω_rel = 48 + 48 = 96°/s
正对间隔 = 36°/96°/s = 0.375s  ← 每0.375秒一对胶囊正对
```
加上偏航追踪±30°，持续正对可维持。

## 3. 动力学与功率

### 3.1 升降驱动

**内环：**
```
F_net = (18,300 - 11,900) × 9.81 = 62,784 N ≈ 62.8 kN
P_normal = 62.8 × 2 / 0.88 = 142.7 kW  (v=2m/s)
P_burst  = 62.8 × 4 / 0.88 = 285.5 kW  (v=4m/s, 短时)
```

**外环：**
```
F_net = (24,700 - 16,100) × 9.81 = 84,366 N ≈ 84.4 kN
P_normal = 84.4 × 2 / 0.88 = 191.8 kW  (v=2m/s)
P_burst  = 84.4 × 4 / 0.88 = 383.6 kW  (v=4m/s, 短时)
```

**结论：内环160kW / 外环200kW。** 分别选型，外环终局4m/s需短时过载至192%，可接受。

### 3.2 旋转驱动

**转动惯量 (环+悬臂+胶囊, 简化为薄环):**

内环：I_inner ≈ 18,300 × 6² = 658,800 kg·m²
外环：I_outer ≈ 24,700 × 8² = 1,580,800 kg·m²

加速段 (0→8rpm in 10s, α=0.0838 rad/s²):

| | 内环 | 外环 |
|---|---|---|
| 加速扭矩 | 658,800×0.0838 = 55,200 N·m | 1,580,800×0.0838 = 132,400 N·m |
| 加速峰值功率 | 55,200×0.838 = 46.3 kW | 132,400×0.838 = 111.0 kW |
| 每组电机功率(÷4) | 11.6 kW → **15kW** | 27.7 kW → **30kW** |

**修正：内环4×15kW, 外环4×30kW。**（原设计未区分内外环差异。）

稳态摩擦功率（回转支承 μ≈0.005）可忽略（<2kW）。

### 3.3 胶囊驱动

**偏航 (Yaw, ±30°):**
- 胶囊绕偏航轴惯量: I_yaw ≈ 350 × 1.2² = 504 kg·m²
- α_yaw = 20°/s² = 0.349 rad/s²
- τ_yaw = 504 × 0.349 = 176 N·m
- 行星减速 i=50, η=0.9: 电机扭矩 = 176/(50×0.9) = 3.9 N·m
- **750W伺服 OK** (额定~3.6 N·m, 瞬时过载可行)

**俯仰 (Pitch, ±30°):**
- 同前计算: τ_pitch ≈ 779 N·m (含重力偏心力矩)
- 蜗轮蜗杆 i=120, η=0.7: 电机扭矩 = 779/(120×0.7) = 9.3 N·m
- **1.5kW伺服 OK** (额定~7.2 N·m @2000rpm, 瞬时过载×1.5)

## 4. 能量分析

### 4.1 双环提升能量

战斗阶段从20m→35m (幅度15m, 往返):

```
内环: E_inner = 18,300 × 9.81 × 15 = 2,692,845 J ≈ 2.69 MJ
外环: E_outer = 24,700 × 9.81 × 15 = 3,634,605 J ≈ 3.63 MJ
```

计入配重回收:
```
实际消耗(单向) :
  内环 = 2.69 × 0.35 / 0.88 = 1.07 MJ
  外环 = 3.63 × 0.35 / 0.88 = 1.44 MJ
往返合计 ≈ (1.07+1.44)×2 = 5.02 MJ ≈ 1.39 kWh
```

**双环单次运行总能耗 ≈ 2.0 kWh** (含旋转+胶囊驱动)。人均电费 ≈ 0.04元。

## 5. 结构校核 (关键项)

### 5.1 钢缆选型

**外环(最不利) 3缆工况:**
- 单缆受力 = 84,400/3 × 1.5(动载) = 42,200 N
- φ28mm 破断拉力 764 kN
- **安全系数 = 764/42.2 = 18.1** ≥ 10 ✅

### 5.2 塔体屈曲

轴向总载荷 (双环满载+配重+塔自重):
```
F_axial = (18,300+24,700 + 11,900+16,100) × 9.81 + 88,000×9.81
        = 71,000×9.81 + 88,000×9.81
        = 1,560 kN
```

I_total ≈ 0.213 m⁴ (同前), L=50m 悬臂柱:
```
P_cr = π²×206e9×0.213 / (4×50²) = 4,330 kN
安全系数 = 4,330/1,560 = 2.78  ≥ 2.5 ✅
```

### 5.3 外环悬臂校核 (φ16m, 长5.0m, 最不利)

```
悬臂 □350×250×12, W_x ≈ 1.2×10⁶ mm³
末端载荷 = 350kg × 9.81 × 5.0m × 2.5(动载) = 42,919 N·m
σ = 42,919e3 / 1.2e6 = 35.8 MPa
安全系数 = 345/35.8 = 9.6  ✅
```

### 5.4 风载挠度

塔顶侧移 42mm, 挠跨比 1/1190 ≤ 1/500 ✅
