## 2. 运动学计算

### 2.1 坐标系回顾

```
全局坐标系 O-XYZ（地面参考系）：
  Z ↑ 垂直向上，原点在地面塔底中心
  θ 绕Z轴旋转角（逆时针为正）

环体位置：
  z(t)   — 环体高度（滑架在塔上的位置）
  θ(t)   — 环体旋转角
  β_i(t) — 第i个胶囊的俯仰角

胶囊i在环上的固定角度：α_i = 36° × (i-1), i=1..10
```

### 2.2 胶囊舱的绝对运动

胶囊舱 i 在全局坐标系中的位置（取胶囊底板中心为参考点）：

```
x_i(t) = R·cos(θ(t) + α_i)                           ... (1)
y_i(t) = R·sin(θ(t) + α_i)                           ... (2)
z_i(t) = z(t) + L·sin(β_i(t))                        ... (3)

其中：R = 7m（环半径）, L = 2.5m（悬臂长+胶囊半高）
```

**速度（对时间求一阶导）：**

```
v_xi = -R·ω(t)·sin(θ + α_i)                          ... (4)
v_yi =  R·ω(t)·cos(θ + α_i)                          ... (5)
v_zi = v_z(t) + L·ω_βi·cos(β_i)                     ... (6)

其中：ω = dθ/dt, v_z = dz/dt, ω_βi = dβ_i/dt
```

**加速度（对时间求二阶导）：**

```
a_xi = -R·α(t)·sin(θ+α_i) - R·ω²(t)·cos(θ+α_i)      ... (7)
        \______切向______/   \_______向心_______/

a_yi =  R·α(t)·cos(θ+α_i) - R·ω²(t)·sin(θ+α_i)      ... (8)

a_zi = a_z(t) + L·α_βi·cos(β_i) - L·ω_βi²·sin(β_i)  ... (9)
        \_________升降_________/   \____俯仰贡献_____/
```

### 2.3 运行工况参数

| 阶段 | z范围 | v_z | ω_rot | β范围 | ω_β | 时长 |
|------|-------|-----|-------|-------|-----|------|
| 上下客 | 5m（固定） | 0 | 0 | 0° | 0 | ~90s |
| 爬升预热 | 5→20m | 0.5 m/s匀速 | 0→6 rpm(加速) | ±5°缓动 | ≤5°/s | ~30s |
| 战斗阶段 | 20↔40m | ±2 m/s正弦 | 6-8 rpm | ±15°联动 | ≤20°/s | ~90s |
| 终局胜方 | 40→48m | +4 m/s | 递减 | +20°(拉升) | ≤30°/s | ~8s |
| 终局败方 | 40→8m | -4 m/s | 递减 | -25°(俯冲) | ≤30°/s | ~8s |
| 返航 | →5m | -1 m/s | →0 rpm | →0° | — | ~30s |

### 2.4 乘客加速度包络计算

**工况A：战斗阶段正常工况（最大组合）**

环旋转角速度 ω = 8 rpm = 0.838 rad/s

向心加速度（环外缘）：
```
a_n = ω²R = (0.838)² × 7.0 = 4.91 m/s² = 0.50g  ← 水平面内
```

垂直加速度（正弦升降，幅度A=10m, 周期T=45s, 即半程22.5s, ω_z = 2π/45 = 0.140 rad/s）：
```
z(t) = 30 + 10·sin(0.140t)    [30m为中位高度]
v_z(t) = 10 × 0.140 × cos(0.140t) = 1.40·cos(0.140t) m/s
a_z(t) = -10 × 0.140² × sin(0.140t) = -0.196·sin(0.140t) m/s²

峰值 |v_z| = 1.40 m/s ✓ (< 2 m/s 战斗工况上限)
峰值 |a_z| = 0.196 m/s² = 0.02g  ← 垂直加速度很小（舒适）
```

战斗阶段最大组合加速度（乘客感受）：
```
a_combat = √(a_n² + (g ± a_z)²)
         = √(0.50² + 1.00²)    [取a_z≈0]
         = 1.12g
```
> 结论：战斗阶段正常工况下乘客加速度约 **1.1g**，非常舒适。

**工况B：终局冲顶（胜方）**

从40m以4m/s拉升到48m，约2秒：
```
a_z_avg = Δv/Δt = 4/2 = 2 m/s² = 0.20g（向上加速）

叠加重力：乘客感受 = g + 0.20g = 1.20g ↓（超重）
```

**工况C：终局急降（败方）**

从40m以4m/s下冲到8m，约8秒。初段加速：
```
取加速段0→4m/s用1.5秒：
a_z = 4/1.5 = 2.67 m/s² = 0.27g（向下加速）

乘客感受 = g - 0.27g = 0.73g（部分失重，约73%体重感）
```

持续减速至8m处停止，减速段约2秒：
```
a_z_brake = 4/2 = 2 m/s² = 0.20g（向上减速）

乘客感受 = g + 0.20g = 1.20g ↓（超重制动）
```

**工况D：紧急制动**

最严酷工况——战斗阶段全速（v_z=2m/s, ω=8rpm）紧急停止：
```
垂直制动距离 ≤ 4m（按GB 8408）：
a_z_brake = v²/(2s) = 2²/(2×4) = 0.5 m/s²  ← 制动减速度较小

但考虑旋转+制动的组合：
旋转制动在2秒内从8rpm→0：
α_rot = 0.838/2 = 0.419 rad/s²
a_tan = α × R = 0.419 × 7 = 2.93 m/s² = 0.30g（切向减速）

组合：√(0.30² + 0.05²) ≈ 0.30g  ← 仍很安全
```

### 2.5 加速度汇总

| 工况 | 垂向加速度 | 径向加速度 | 组合峰值 | GB 8408限值 | 判定 |
|------|-----------|-----------|---------|-------------|------|
| 正常战斗 | ±0.02g | 0.50g | **~1.1g** | 4g | ✅ |
| 终局冲顶 | +0.20g | 渐减 | **~1.2g** | 4g | ✅ |
| 终局急降 | -0.27g | 渐减 | **~0.7g** (失重) | 4g | ✅ |
| 急降制动 | +0.20g | — | **~1.2g** | 4g | ✅ |
| 紧急制动 | +0.05g | 0.30g | **~1.0g** | 4g (瞬时6g) | ✅ |

> **关键结论**：所有工况乘客加速度均 ≤1.2g，远低于 GB 8408 的 4g 限值。设计偏保守，**以俯卧姿态的失重体验为核心刺激**，而非单纯堆G值。

## 3. 动力学与功率计算

### 3.1 升降驱动功率

**设计参数：**
- 满载环重：M_load = 21,300 kg
- 配重比：65% → M_cw = 13,845 kg
- 钢缆系统效率：η = 0.88（含滑轮组摩擦）
- 卷筒直径：D_drum = 1.2m

**不平衡力（配重后）：**
```
F_net = (M_load - M_cw) × g = (21,300 - 13,845) × 9.81
      = 7,455 × 9.81 = 73,130 N ≈ 73.1 kN
```

**正常升降（v = 2 m/s）：**
```
P_normal = F_net × v / η = 73,130 × 2 / 0.88
         = 166,200 W ≈ 166 kW
→ 选用 160kW 变频电机（短时过载能力150%可覆盖）
```

**终局急升（v = 4 m/s, 短时 ≤10s）：**
```
P_burst = 73,130 × 4 / 0.88 = 332,400 W ≈ 332 kW
→ 160kW 电机过载至208%运行 8-10秒（需验证电机热容量）
→ 或选用 200kW 电机（更安全）
```

**结论：升降电机调整为 2 × 200kW（每环一台）**，装机总功率相应调整。

### 3.2 旋转驱动功率

**环体转动惯量（近似为薄圆环）：**
```
I_ring = M_load × R² = 21,300 × 7² = 1,043,700 kg·m²
```

加上悬臂和胶囊的附加惯量（平行轴定理）：
- 每个胶囊+悬臂质量约 350kg, 距中心 7m
- 附加：10 × 350 × 7² = 171,500 kg·m²
- **总转动惯量 I_total ≈ 1,215,000 kg·m²**

**加速段（0→8 rpm 用 10s）：**
```
ω_final = 8 × 2π/60 = 0.838 rad/s
α = 0.838/10 = 0.0838 rad/s²
τ_acc = I × α = 1,215,000 × 0.0838 = 101,800 N·m
P_acc_peak = τ_acc × ω_final = 101,800 × 0.838 = 85,300 W ≈ 85 kW
```

**稳态运行（仅克服摩擦）：**
滚动摩擦系数 μ ≈ 0.005（回转支承滚柱）:
```
τ_friction = μ × M_load × g × R_bearing
           = 0.005 × 21,300 × 9.81 × 1.75  [R_bearing=3.5/2=1.75m]
           = 1,828 N·m
P_steady = 1,828 × 0.838 = 1,532 W ≈ 1.5 kW  ← 极低
```

**结论：旋转驱动 4 组，每组 22kW → 总计 88kW。** 正常运行时仅需克服摩擦（极小功率），加速段短时功率由电机过载能力覆盖。

### 3.3 胶囊俯仰驱动

俯仰轴惯量（胶囊+2人绕俯仰铰）：
- 胶囊满载质量 ~350kg, 绕铰点回转半径 ~1.2m
- I_pitch ≈ 350 × 1.2² = 504 kg·m²
- 最大角加速度：α_β = 30°/s² = 0.524 rad/s²
- τ_pitch = 504 × 0.524 = 264 N·m
- 加上重力偏心力矩（重心偏离铰点约0.3m）：350 × 9.81 × 0.3 × sin(30°) = 515 N·m
- **最大力矩 ≈ 264 + 515 = 779 N·m**
- 蜗轮蜗杆减速比 i=80, 效率 0.7: 电机扭矩 = 779/(80×0.7) = 13.9 N·m
- **选取 1.5kW 伺服电机**（额定扭矩 7.2 N·m @2000rpm, 经 i=80 输出 403 N·m → 需更大的 i 或选更大电机）
- 调整为 **i=120**, 则电机扭矩 = 779/(120×0.7) = 9.3 N·m → 1.5kW 可满足（瞬时过载）
- 稳态保持用蜗轮蜗杆自锁（导程角 < 摩擦角）

## 4. 能量分析

### 4.1 单次运行能量消耗

以战斗阶段为核心（忽略上下客和爬升的小能量）：

**提升重力势能（战斗阶段从20m→40m再返回）：**
```
单环升降一个循环：
Δh = 20m（上下幅度）
E_pot = M_load × g × Δh = 21,300 × 9.81 × 20 = 4,179,060 J ≈ 4.18 MJ

计入配重回收：配重下降回收能量 ≠ 100%效率
实际消耗 ≈ E_pot × (1 - 0.65) × (1/η) = 4.18 × 0.35 / 0.88 ≈ 1.66 MJ（单向）
往返 ≈ 3.32 MJ
```

**旋转能量（战斗阶段 90s）：**
```
稳态摩擦功率 1.5kW × 90s = 135 kJ ← 可忽略
加速段能量 ≈ 85kW × 10s / 2 = 425 kJ（三角形功率曲线）
往返各加速一次 ≈ 850 kJ
```

**双环单次运行总能耗：**
```
E_total ≈ 2 × (3.32 + 0.85) = 8.34 MJ ≈ 2.32 kWh
```

以商业电价约 0.8 元/kWh 计，单次运行电费约 **1.86 元**。载客 40 人，人均电费约 **0.05 元**。电力成本可忽略。

### 4.2 能量回收潜力

终局急降时，败方环从 40m→8m 释放势能：
```
ΔE = 21,300 × 9.81 × 32 = 6,684,096 J ≈ 6.68 MJ
```
此能量如通过变频器回馈电网（再生制动），可回收约 60-70%，供给同期胜方环冲顶所需能量。

## 5. 关键结构校核（初步）

### 5.1 钢缆选型

**载荷条件：**
- 每环 4 根钢缆（安全冗余：1根断裂仍可安全制动）
- 3根缆工况（最不利）：单缆受力 = 73,130 / 3 = 24,377 N
- 加上动载系数 1.5：F_design = 24,377 × 1.5 = 36,566 N

**选型：φ28mm, 6×36WS-IWRC（钢芯）, 公称抗拉强度 1960 MPa**
- 金属截面积 ≈ 390 mm²
- 最小破断拉力 F_min = 390 × 1960 = 764,400 N ≈ 764 kN
- **安全系数 = 764/36.6 = 20.9**（远大于 GB 8408 要求的 10）

✅ 安全裕度充足。

### 5.2 塔体稳定性（欧拉屈曲校核）

塔体简化为悬臂柱：高 50m, □3m×3m 四肢桁架。

**截面惯性矩估算（四肢主管 + 平行轴定理）：**
- 单根主管 φ610×20, A = π(610×20 - 20²) ≈ 37,100 mm²
- 单根 I_self = π/64 × (610⁴ - 570⁴) ≈ 3.3 × 10⁸ mm⁴
- 平行轴：d ≈ 1.5 - 0.305 = 1.195m（主管中心到截面中心）
- I_total = 4 × (3.3×10⁸ + 37,100 × 1195²)
- = 4 × (3.3×10⁸ + 5.30×10¹⁰) ≈ 4 × 5.33×10¹⁰ = 2.13×10¹¹ mm⁴ = 0.213 m⁴

**临界屈曲载荷：**
```
P_cr = π²EI/(4L²)    [悬臂柱，有效长度为2L]
     = π² × 206×10⁹ × 0.213 / (4 × 50²)
     = 3.14² × 206×10⁹ × 0.213 / 10,000
     = 43,294 × 10⁶ / 10,000
     = 4.33 × 10⁶ N = 4,330 kN = 441 t
```

**实际最大轴力：**
```
塔顶载荷 = 双环满载 + 配重 = 2 × 21,300 + 2 × 13,845 = 70,290 kg
塔体自重 = 88,000 kg
合计 ≈ 158,290 kg → 轴向力 = 1,553 kN

屈曲安全系数 = 4,330 / 1,553 = 2.79
```

✅ 整体稳定性满足（安全系数 > 2.5）。但注意此计算简化了缀条效应，更精确的分析需在 ANSYS 中进行线性屈曲分析（Buckle Analysis）。

### 5.3 塔顶挠度估算

水平风载下的塔顶侧移（简化计算，风载按均布荷载近似）：

风压（12级风, v=32.7m/s）：
```
q_wind = 0.5 × ρ_air × C_d × v² × B
       = 0.5 × 1.225 × 1.2 × 32.7² × 3.0    [B=塔宽3m]
       = 2,356 N/m
```

悬臂柱顶点挠度：
```
δ_max = q_wind × L⁴ / (8EI)
      = 2,356 × 50⁴ / (8 × 206×10⁹ × 0.213)
      = 2,356 × 6,250,000 / (8 × 206×10⁹ × 0.213)
      = 1.47×10¹⁰ / 3.51×10¹¹
      = 0.042 m = 42 mm
```

**挠跨比 = 42/50,000 = 1/1190** ≤ 1/500（GB 8408 限值） ✅ 刚度充足。

### 5.4 悬臂支架校核

悬臂长 3.5m, 末端载荷 = 胶囊满载重量/2 = 350×9.81/2 = 1,717 N（每个悬臂两个铰点）。
取动载系数 2.5（紧急制动）：

```
M_max = F × L × 2.5 = 1,717 × 3.5 × 2.5 = 15,024 N·m

□300×200×10 矩形管截面模量（近似）：
W_x ≈ (200×300² - 180×280²)/6 ≈ 9.0×10⁵ mm³

σ = M/W = 15,024×10³ / 9.0×10⁵ = 16.7 MPa
```

Q345 钢材 σ_s = 345 MPa, 安全系数 = 345/16.7 = **20.7** ✅ 极安全（变形控制设计，非强度控制）。

## 6. 制动距离验证

最严酷制动工况：终局急降时 4m/s → 0, 要求在 4m 内完成。

```
v₀ = 4 m/s,  v_f = 0,  s = 4m
a_required = v₀²/(2s) = 16/8 = 2 m/s² = 0.20g
```

制动系统需要提供 2 m/s² 的制动力，加上重力：
```
F_brake = M_load × (a_required + g) = 21,300 × (2 + 9.81) = 251,553 N

分配到4组轨刹：每组 62,888 N ≈ 63 kN
```
弹簧蓄能-液压释放式轨刹可轻松提供此级别的制动力。

✅ 制动距离 4m 可实现。
