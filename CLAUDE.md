## 当前方案：「天袭中队」双环竞技跳楼机

核心概念：中央塔悬挂两个独立环（红队/蓝队），每环10个双人胶囊舱，总载客40人/次。双环反向旋转+独立升降，胶囊俯仰联动。两队激光对射竞技，胜方环冲顶（奖励），败方急降（惩罚）。

创新点：双环耦合运动 + 编队竞技 + 俯卧战斗姿态 + 竞技结果决定物理体验。

## 设计流程（四阶段）

这是本项目的核心工作流。**每个阶段完成后需与用户确认再进入下一阶段。**

### Phase 1: 概念设计

> 确定"做什么"——主题、体验、核心创新点。

- 选题确认（赛题一/二/三/四）
- 核心体验定义（运动形式、刺激类型、多感官要素）
- 目标参数范围（高度、速度、载客量、G值级别）
- 多个概念方案生成与筛选
- 参考案例调研（已有PPTX中40+全球案例）

### Phase 2: 方案设计（← 当前阶段）

> 确定"怎么做"——**所有关键参数必须有物理/力学计算支撑，不可随意取值。**

**必须先完成计算，再开始建模的科目：**

| 计算领域 | 内容 | 决定的设计参数 |
|----------|------|---------------|
| 质量估算 | 各部件质量、总载荷 | 驱动功率、结构截面 |
| 运动学 | 位移/速度/加速度方程、包络分析 | 轨道曲线、运动范围、乘客G值 |
| 动力学 | 力/力矩/功率需求 | 电机选型、传动比、制动能力 |
| 能量分析 | 势能↔动能转换、能耗 | 配重比、升降速度、回馈策略 |
| 钢缆校核 | 破断拉力、安全系数 | 钢缆直径、数量 |
| 稳定性 | 屈曲、倾覆、挠度 | 塔体截面、基础尺寸 |
| 制动 | 制动距离、减速度 | 制动器类型与规格 |

**输出文件：**
- `设计方案_天袭中队.md` — 总体方案、机构原理、技术参数
- `工程计算书_天袭中队.md` — 全部计算过程与校核结论

**阶段完成标志：** 所有关键尺寸和参数均有计算依据，用户确认通过。

### Phase 3: 详细设计

> SolidWorks 分模块建模 + ANSYS/ADAMS 仿真验证。

**建模顺序（按依赖关系）：**
1. 地基基础
2. 中央塔体（Q345钢管桁架, □3m×3m, H=50m）
3. 升降滑架（含导向滚轮组）
4. 回转支承（三排滚柱式 φ3.5m）
5. 环形桁架（φ14m）
6. 悬臂支架 ×10（□300×200×10, L=3.5m）
7. 胶囊舱主体 ×10（铝合金+玻璃, 双人俯卧）
8. 安全约束系统（五点式安全带+腿部压杆）
9. 俯仰驱动机构（伺服+蜗轮蜗杆）
10. 钢缆+卷扬系统（φ28mm, 4缆/环）
11. 站台平台

最后：总装配体。

**仿真验证：**
- ANSYS 静力学：6个载荷工况（详见计算书 §7.4）
- ADAMS 动力学：5个仿真工况（详见计算书 §7.3）

### Phase 4: 设计输出

- 设计说明书撰写（四部分，按赛事官方提纲）
- KeyShot 渲染（夜景科幻战斗风格）
- PPT 制作 + 汇报视频录制
- 关键工程图导出

## 核心工作原则

1. **计算先于建模** — 任何尺寸/速度/功率必须有物理计算支撑，不能"大概"。
2. **交互确认** — 每个Phase的关键决策需用户确认后再继续。
3. **大学生比赛水平** — 计算到基本物理/力学层面即可，不过度深入（如不做CFD、不做非线性有限元）。
4. **可追溯** — 设计参数变化时同步更新计算书和方案文档。
5. **分模块工作** — 每完成一个模块的建模/计算，确认后再进入下一个。

## 设计说明书必需内容（四部分）

1. **总体情况**：设施描述、形式特点、主要参数（游玩人数、场地尺寸、设备尺寸、运行速度/加速度范围），配图
2. **创新内容**：结构/功能/造型/运动形式/互动/新技术方面的创新 + 设计计算 + 安全防护 + 可靠性说明
3. **游玩方式**：游玩过程描述、适用场景（室内/室外）、适用对象、乘坐方式
4. **效果评价**：自我评价、与国内外现有设施对比的创新、体验效果、推广前景

## 国家标准要点

- GB 8408-2018《大型游乐设施安全规范》：安全设计总纲
- GB/T 20306-2017《游乐设施术语》：术语定义
- 大型游乐设施定义：最大运行线速度 ≥ 2m/s 或运行高度 ≥ 2m
- 赛题一目标群体：15-25岁青年
- 乘客加速度限值：正常运行 ≤4g，瞬时峰值 ≤6g
- 钢缆安全系数 ≥10
- 结构挠跨比 ≤1/500
