# 天袭中队 — 双环竞技跳楼机 设计方案 v2

> 赛题一：「青年刺激」大型游乐设施创意设计
> 目标群体：15-25岁青年

---

## 1. 总体概述

### 1.1 设施简介

"天袭中队"是一款融合**编队竞技**与**自由落体**的新型大型游乐设施。核心结构为一座50m中央塔，悬挂**内外两个同心环体**，内环（红队，φ12m）与外环（蓝队，φ16m）在同一高度对峙。每环承载10个双人胶囊舱（共40人/次），每舱两人分工——射手负责激光进攻，驾驶员操控舱体闪避。两环反向旋转，竞技积分自动触发弱者补偿机制，终局胜方冲顶、败方急降。

### 1.2 核心创新

- **同心双环**：内外环同高度面对面，最近接敌距离仅2m，双方永远对视
- **角色分工**：射手（进攻）+ 驾驶员（防守），每人都参与竞技
- **弱者补偿**：分差自动调节环体高度，比赛始终胶着
- **竞技决定体验**：胜败不是数字——而是身体感受到的冲顶/急降

### 2.3 加速度包络

| 工况 | 加速度 | 方向 |
|------|--------|------|
| 战斗阶段(8rpm) 向心加速度 | ~0.34g (内环) / ~0.46g (外环) | 径向 |
| 终局急降 | -0.27g | 垂直（部分失重） |

### 3.3 胶囊舱与环体连接（铰接机构）

胶囊舱通过**两级铰接机构**连接于悬臂末端，实现偏航+俯仰2自由度。

```
                    侧视图（沿径向看悬臂末端）
    
        悬臂末端
        □300×200(内) / □350×250(外)
           │
    ╔══════╪══════╗
    ║ ┌────┴────┐ ║
    ║ │ 偏航轴承 │ ║  ← 垂直轴
    ║ │  (Yaw)  │ ║     推力球轴承 + 深沟球轴承
    ║ │ ±30°    │ ║     750W AC伺服 + 行星减速 i=50
    ║ │ 舵机驱动 │ ║     失电制动器
    ║ └────┬────┘ ║
    ║      │      ║
    ║ ┌────┴────┐ ║  ← U型偏航框架 (铸钢)
    ║ │         │ ║
    ║ │  ┌──┐   │ ║
    ║ │  │俯│   │ ║  ← 水平俯仰轴
    ║ │  │仰│   │ ║     双列角接触球轴承
    ║ │  │轴│   │ ║     1.5kW伺服 + 蜗轮蜗杆 i=120 (自锁)
    ║ │  └──┘   │ ║
    ║ │         │ ║
    ║ └────┬────┘ ║
    ╚══════╪══════╝
           │
    ┌──────┴──────┐
    │   胶囊舱     │
    │  连接法兰    │
    └─────────────┘


                    俯视图
    
        悬臂 ──────────→ 径向
    
          ┌─────────┐
          │  偏航轴  │  ← ±30°, 追踪对手
          │   (↔)   │
          └────┬────┘
               │
          ┌────┴────┐
          │ U型框架  │
          │┌──┐ ┌──┐│
          ││轴│ │轴││  ← 俯仰轴两端支撑
          ││承│ │承││     悬臂长度使其有足够空间
          │└──┘ └──┘│
          │  胶囊舱  │
          │  连接点  │
          └─────────┘
```

**关节顺序逻辑**：偏航在上 → 俯仰在下。偏航旋转时把整个俯仰机构+胶囊一起转，俯仰轴始终与胶囊切向垂直，运动解耦。

**失效保护**：
- 断电 → 偏航制动器弹簧锁死 + 俯仰蜗轮自锁 → 舱体固定在当前位置
- 系统接管 → 缓慢回中 → 手动解锁

### 3.5 安全约束系统（俯卧式）

- 五点式背部安全带（肩胛→腰部，带预紧器）
- 腿部液压压杆（大腿+小腿）
- 脚踝可调式脚踏+绑带
- 所有约束带位置传感器，任一未锁→环体禁止启动

## 4. 驱动与传动系统

### 4.1 升降驱动

- 每环独立卷扬机：200kW变频AC电机 + 行星减速器 + φ1.2m卷筒
- 每环4根φ28mm钢缆（6×36WS-IWRC, 破断≥764kN, 安全系数≥20）
- 配重比65%，配重块~14t（混凝土+钢壳）
- 变频调速 0-4m/s 无级

### 4.2 旋转驱动

- 三排滚柱式回转支承（φ3.5m, 外齿圈）
- 每环4组驱动小齿轮（22kW伺服 + 行星减速 i=50）
- 两环反向旋转（内环↻, 外环↺）

### 4.3 胶囊驱动

| 关节 | 电机 | 减速器 | 自锁 | 安全 |
|------|------|--------|------|------|
| 偏航 | 750W AC伺服 | 行星 i=50 | ❌ | 失电制动器 |
| 俯仰 | 1.5kW AC伺服 | 蜗轮蜗杆 i=120 | ✅ | 自锁+失电制动器 |

## 5. 竞技系统

### 5.1 面对面机制

- 内环胶囊**径向朝外**、外环胶囊**径向朝内**→ 天然面对面
- 两环反向旋转（相对角速度~96°/s）→ 每0.375秒有一对胶囊正对
- 驾驶员偏航±30°追踪 → 持续正对
- 2m透明舱面对面 → 可看到对手表情，刺激传染

### 5.2 激光对射

- 每舱2套激光发射器（左右握把扳机）+ 4个接收器（前后左右）
- 激光编码调制区分红蓝队
- 命中=短震+音效, 被命中=舱体前倾2°模拟中弹

### 5.3 弱者补偿机制

```
        48m ──┐                    ← 高空 = 进攻优势
              │   "高空优势区"        命中+20%加分
        35m ──┤   但被弹面积更大      (更容易被打中)
              │
              │   "均衡交战区"       正常得分
        20m ──┤   双方持平
              │
              │   "低空掩护区"       被弹面积小
         8m ──┤   但命中-20%减分
              │
         5m ──┘   ← 站台
```

| 分差 | 系统干预 |
|------|---------|
| ≤20分 | 不干预，纯技术对抗 |
| 20-50分 | 落后方环 ±5m 高度补偿 |
| >50分 | 落后方升入高空优势区，领先方降至低空 |

终局不受补偿影响——结束前30秒锁定高度，打决胜局。

### 5.4 运行流程

| 阶段 | 时长 | 高度&运动 | 竞技 |
|------|------|----------|------|
| 上下客 | ~90s | 5m, 静止 | 激光校验 |
| 爬升预热 | ~30s | 5→20m, 加速 | 练习射击（不计分） |
| 战斗阶段 | ~90s | 20-35m正弦升降+旋转 | **正式积分**, 实时补偿 |
| 终局 | ~15s | 胜方→48m冲顶 / 败方→8m急降 | 锁定 |
| 返航 | ~30s | 缓降至5m | 显示排名 |
