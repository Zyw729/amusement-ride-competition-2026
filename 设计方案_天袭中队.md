# 天袭中队 — 双环竞技跳楼机 设计方案

> 赛题一：「青年刺激」大型游乐设施创意设计
> 目标群体：15-25岁青年

---

## 1. 总体概述

### 1.1 设施简介

"天袭中队"是一款融合**编队竞技**与**自由落体**的新型大型游乐设施。核心结构为一座50m中央塔，悬挂红蓝两个独立环体，每环承载10个双人胶囊舱（共40人/次）。两环反向旋转、独立升降，乘客以俯卧姿态进行激光对射竞技，竞技结果直接决定终局体验——胜方冲顶奖励，败方急降惩罚。

## 2. 技术参数汇总

### 2.1 总体参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 设备总高 | 50.0 m | 塔顶平台高程 |
| 环体直径 | 14.0 m | 环形桁架外径 |
| 占地面积 | φ22 m | 含安全围栏 |
| 额定载客 | 40 人/次 | 双环 × 10舱 × 2人 |
| 理论小时容量 | ~480 人/h | 按单次运行5分钟（含上下客） |
| 设备等级 | A级 | 运行高度≥2m且速度≥2m/s |
| 设计寿命 | 15 年 | 主体钢结构 |

### 2.2 运动参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 环体升降行程 | 43 m | 最低5m（上下客）→ 最高48m |
| 最大升降速度 | 4.0 m/s (14.4 km/h) | 终局冲顶/急降段（经功率校核） |
| 正常运行升降速度 | 2.0 m/s | 爬升/战斗阶段 |
| 环体转速 | 0~10 rpm | 可调，战斗阶段6-8 rpm |
| 最大切线速度 | ~7.3 m/s (26 km/h) | 环体外缘（r=7m, 10rpm） |
| 胶囊俯仰范围 | ±30° | "俯冲""拉升"姿态 |
| 胶囊俯仰角速度 | ≤30°/s | 伺服驱动 |

### 2.3 加速度范围（设计值）

| 工况 | 加速度 | 方向 |
|------|--------|------|
| 正常升降（战斗阶段） | ±0.02g | 垂直 |
| 终局急降 | -0.27g | 垂直（部分失重） |
| 终局冲顶/急降制动 | +0.20g | 垂直（超重） |
| 环体旋转（8rpm） | ~0.50g | 径向（离心） |
| 胶囊俯仰 | ±0.2g | 切向 |
| 紧急制动 | ~0.3g | 组合 |
| 所有工况组合峰值 | **≤1.2g** | 垂向+径向叠加 |

> 设计偏舒适保守（峰值仅1.2g），以俯卧姿态的失重感和高度恐惧为核心刺激，而非堆G值。远低于GB 8408的4g限值。详见工程计算书。

### 2.4 电力参数

| 参数 | 数值 |
|------|------|
| 总装机功率 | ~631 kW（经计算修正） |
| 红/蓝环升降电机 | 2 × 200 kW（变频交流, 65%配重） |
| 红/蓝环旋转电机 | 8 × 22 kW（4组×2环, 变频同步） |
| 胶囊俯仰伺服 | 20 × 1.5 kW |
| 控制系统 + 照明 + 激光 | ~25 kW |

**塔体设计要点：**
- 主管：4根 Q345 φ610×20mm 无缝钢管，四角布置
- 水平缀条：φ245×10mm，间距 3m
- 斜向缀条：φ245×10mm，交叉布置（K型或X型）
- 总用钢量估算：~85 吨（塔体 + 平台）
- 防腐：热浸镀锌 + 氟碳面漆（户外25年防护）

### 3.4 安全约束系统

| 约束装置 | 位置 | 类型 |
|----------|------|------|
| 背部主安全带 | 肩胛→腰部 | 五点式安全带，带预紧器 |
| 腿部约束 | 大腿 + 小腿 | 液压锁紧压杆 |
| 脚踝固定 | 脚踝 | 可调式脚踏 + 绑带 |
| 握把 | 双手 | 集成激光扳机 + 紧急按钮 |

**安全联锁：**
- 所有约束装置配备位置传感器
- 任一约束未锁紧 → 环体禁止启动
- 运行中传感器持续监测，异常→紧急制动

## 5. 制动与安全系统

### 5.1 制动层级

| 层级 | 触发条件 | 制动方式 | 制动距离 |
|------|----------|----------|----------|
| 正常工作制动 | 运行结束/降速 | 变频电机再生制动 + 机械盘刹 | 按运行曲线控制 |
| 快速制动 | 风速超限/系统预警 | 涡流制动 + 机械盘刹 | ≤8m |
| 紧急制动 | 安全联锁触发 | 弹簧蓄能轨刹 + 涡流制动 | ≤4m |
| 终极制动 | 钢缆全断（极端工况） | 轨道机械棘轮止挡 | 有限行程 |

### 5.2 监测系统

| 监测项 | 传感器 | 报警阈值 |
|--------|--------|----------|
| 风速 | 塔顶风速仪 | >15m/s 限速运行，>25m/s 停止 |
| 钢缆张力 | 销轴式力传感器（每缆） | 单缆张力偏离均值>30%报警 |
| 环体位置 | 激光测距 + 编码器双冗余 | 位置偏差>100mm 急停 |
| 环体速度 | 编码器微分 + 独立速度传感器 | 超速115% 紧急制动 |
| 约束状态 | 每座微动开关 | 任一未锁 → 禁止启动 |
| 电机温度 | PT100热电阻 | >130°C 降功率，>155°C 停机 |
| 振动 | 塔顶 + 塔基加速度计 | 异常振动频谱 → 停机检查 |

### 5.3 乘客安全约束

- 俯卧姿态特有的安全考量：
  - 背部五点式安全带（非腰部，因俯卧时腰部受力小）
  - 腿部液压压杆（防止膝部弯曲导致滑出）
  - 所有约束在舱门关闭后自动预紧
  - 运行全程锁死，仅地面站台可解锁

## 6. 竞技系统

### 6.1 激光对射机制

- 每舱配2套激光发射器（左右手握把扳机）
- 每舱配4个激光接收器（舱体前/后/左/右）
- 激光编码调制：区分红队/蓝队，防止互射误判
- 有效射程：全环径范围（最近1.5m，最远~18m）
- 命中判定：接收器收到对方编码信号 + 持续时间≥50ms

### 6.2 积分与反馈

| 事件 | 积分 | 物理反馈 |
|------|------|----------|
| 命中对方 | +10分 | 本方舱短促震动 + 音效 |
| 被命中 | -5分 | 舱体前倾2°（模拟中弹失速） |
| 连续命中（3连击） | +30分 | LED光环闪烁 + 全场广播 |
| 终局胜方 | — | 环体冲顶至48m（+2g短暂拉升） |
| 终局败方 | — | 环体急降至8m（-1.2g 失重） |
| 平局 | — | 双环同步缓降 |

### 6.3 运行流程时序

| 阶段 | 时长 | 环体高度 | 环体旋转 | 竞技 |
|------|------|----------|----------|------|

## 7. 运动学与动力学分析框架

### 7.1 坐标系定义

```
全局坐标系 O-XYZ（固定在地面）：
  O  = 塔底中心，地面高程 0
  Z  = 垂直向上
  X  = 正东方向
  Y  = 正北方向

环体局部坐标系：
  z_red(t)  = 红环高度（随时间变化）
  z_blue(t) = 蓝环高度
  θ_red(t)  = 红环旋转角（绕Z轴）
  θ_blue(t) = 蓝环旋转角（反向）
  
胶囊 i 在环上的位置角：
  α_i = 2π(i-1)/N , N=10, i=1..10
```

### 7.2 运动方程

**红环胶囊舱 i 的全局坐标：**

```
x_i(t) = R·cos(θ_red(t) + α_i)
y_i(t) = R·sin(θ_red(t) + α_i)
z_i(t) = z_red(t) + h_arm·sin(β_i(t))
```

其中：
- R = 7m（环体半径）
- h_arm = 2.5m（悬臂长度）
- β_i(t) = 胶囊俯仰角

**速度方程（对时间求导）：**

```
v_x = -R·ω_red·sin(θ_red + α_i)
v_y =  R·ω_red·cos(θ_red + α_i)
v_z = v_z_ring + h_arm·ω_β·cos(β_i)
```

**加速度方程（二次求导）：**

```
a_径向 = R·ω_red²（离心）
a_切向 = R·α_red（角加速度引起的切线加速度）
a_垂直 = a_z_ring + h_arm·[α_β·cos(β) - ω_β²·sin(β)]
a_科氏 = 2·ω_red·v_z（科里奥利加速度，环旋转+升降耦合）
```

### 7.3 ADAMS 动力学仿真建议

**模型构建：**
- 塔体：固定刚体（Ground）
- 红环滑架：垂直移动副（Translational Joint）
- 蓝环滑架：垂直移动副
- 红环桁架：旋转副（Revolute Joint），基于红环滑架
- 蓝环桁架：旋转副，基于蓝环滑架
- 20个胶囊舱：各一个旋转副（俯仰），基于各自环体

**仿真工况：**

| 工况 | 输入 | 关注输出 |
|------|------|----------|
| 正常战斗 | 双环正弦升降+匀速旋转 | 胶囊加速度包络、乘客G值 |
| 终局冲顶/急降 | 红环5m/s↑, 蓝环5m/s↓ | 峰值加速度、制动距离 |
| 紧急制动 | 全速→制动 | 制动距离、乘客冲击G值 |
| 单缆断裂 | 4缆→3缆 | 偏载量、安全裕度 |
| 风载工况 | 侧风15/25m/s | 环体偏摆、导轨侧向力 |

**关键验证指标（GB 8408）：**
- 乘客加速度 < 4g（正常运行）
- 乘客加速度 < 6g（瞬时、非经常性）
- 紧急制动后乘客无明显不适
- 结构应力 < 材料许用应力/安全系数

## 8. 创新点总结

### 8.1 结构创新

| 创新点 | 说明 | 对比现有设施 |
|--------|------|------------|
| **双环独立运动** | 同一塔体两环独立升降+旋转，全球首创 | 现有跳楼机/大摆锤均为单运动体 |
| **胶囊俯仰联动** | 俯仰角度与竞技事件联动，增强沉浸感 | 现有设施俯仰为固定曲线或随机 |
| **俯卧编队姿态** | 40人同时以"空中编队"战斗姿态体验 | 伊卡洛斯仅16人、无竞技 |

### 8.2 功能创新

| 创新点 | 说明 |
|--------|------|
| **竞技结果决定物理体验** | 胜败差异化——不是"分数显示在屏幕上"，而是"你的身体感受到胜利或失败的G力" |
| **PvP编队对战** | 首次将电子竞技的团队对抗引入大型游乐设施 |
| **社交+复玩驱动** | 每次对战不同（对手、队友不同），天然具有复玩吸引力 |

### 8.3 体验创新

| 创新点 | 说明 |
|--------|------|
| **三段式叙事结构** | 爬升（紧张）→ 战斗（高潮）→ 终局（释放），完整故事线 |
| **多感官同步** | 视觉（透明舱底俯瞰+激光光效）、触觉（G力+震动）、听觉（音效+语音播报） |
| **俯卧视角独特性** | 脸朝下的"飞翔"感 + 俯瞰地面的高度恐惧 |
